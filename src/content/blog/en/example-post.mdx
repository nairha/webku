# Multi-language Support with Next.js

This is a sample post in English. Each language now has its own directory in `src/content/blog/[lang]`.

This ensures that:
1. Content is independent for each language.
2. SEO is improved as each page has unique localized content.
3. No fallback to Indonesian if an English version doesn't exist.
